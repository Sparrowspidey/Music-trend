# Spotify Music Trend Finder

A Python app that lets users explore trending playlists and listen to 30-second song previews using the Spotify API.

---

##  Features
- Search and fetch Spotify playlists by categories:
  - Global
  - India
  - Love
  - Breakup
  - Party
  - Mass
- Show top 5 tracks from selected playlist
- Play 30-second preview using pygame
- Open full track directly in Spotify
- Clean console menu-driven UI
